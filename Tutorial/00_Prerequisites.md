# Prerequisites

### Key

This key is the same as used in the Vim docs

- `<key>`: Key with special name
- `<C-key>`: CTRL+key
- `<S-key>`: Shift+key (only used for special characters)
- `<A-key>`: ALT+key
- `<CR>`: Enter key
- `{thing}`: Required
- `[thing]`: Optional
